Mengunduh seluruh berkas dalam satu folder

wget -r --no-parent url

Melanjutkan unduh berkas

wget -c url
wget --continue url
wget --continue [options] url

Mounting berkas iso

mount -o loop file.iso /media/target_folder

SSH Tunneling

ssh -l user url/ip -D xxxxx
ex: ssh -l root 10.0.0.1 -D 65535

Melihat folder atau berkas dengan ukuran terbesar

du -a /ftpusers/tmp | sort -n -r | head -n 10
du -cks * | sort -rn | head

Menggunakan screen

screen -S namascreen

Melepas screen

ctrl-a+d

Melihat screen

screen -ls

Melanjutkan screen

screen -r namascreen

Melanjutkan screen yang masih terpasang

screen -d namascreen

Rsync dengan sistem yang diremote

rsync -a ~/dir1 user@host:destination_directory
rsync -a user@host:~/dir1 dir_to_sync_on_local
rsync -azP source destination

Menghentikan program berjalan

kill -15 PID
killall -15 firefox

*lakukan jika sudah terpaksa
kill -9
